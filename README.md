# Mission Engine

> AI system that discovers global missions and spawns companies to solve them.

## Setup

```bash
npm install
cp .env.example .env
# Add OPENAI_API_KEY to .env
```

## Run

```bash
npm run dev
```

## v1 Features

1. **Mission Discovery** - Feed of AI/Technology missions
2. **Mission Page** - Problem, Impact, Signals, Status
3. **Generate Solutions** - AI-generated solution paths
4. **Spawn Company** - AI-generated Company Blueprint
