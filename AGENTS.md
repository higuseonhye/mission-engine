# Mission Engine

> AI system that discovers global missions and spawns companies to solve them.

## Project Overview

Mission Engine is the first product of the Civilization Engine vision. It enables users to:
1. **Discover** global missions (problems worth solving)
2. **Explore** solutions and startup ideas
3. **Spawn** company blueprints to execute missions

## v1 Scope

- **Category**: AI / Technology
- **Features**: Mission Feed, Mission Page, Generate Solutions, Spawn Company
- **Goal**: "사람이 이 플랫폼에서 미션을 발견하고 회사를 만들고 싶다"

## Key Files

- `src/app/` - Next.js App Router pages
- `src/lib/` - Data models, API clients
- `src/data/` - Seed missions (first 20)

## Conventions

- TypeScript strict mode
- Tailwind for styling
- Keep v1 minimal - no over-engineering
